# Systemic

Serious game cooperativo para treinamento de times de tecnologia, executado como tabuleiro digital remoto.

## Estrutura
- client/ — Interface web (React + Vite)
- server/ — Servidor autoritativo (Node.js + Socket.IO)
- docs/ — Documentação técnica e de projeto

## Execução local (desenvolvimento)
- client: `npm run dev`
- server: `npm run start`

## Execução em produção
O build do client é servido pelo server e implantado como Web Service no Render.
